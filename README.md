# BST-BRDI-Calculator
## 概要
[さんらいく氏](https://twitter.com/fallen_helga)考案の難易度表記を、公式サイトから引っ張ってきて自動的に表示させるものです。

## 使い方
cronをいい感じに叩いてください。
